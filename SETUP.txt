💾 BANCO DE DADOS
═══════════════════════════════════════════════════════════════════════════════

Banco:     ControleExterno
Tabela:    Users
Host:      127.0.0.1
Porta:     3306
Usuário:   root
Senha:     (vazio)

CAMPOS DA TABELA:
   • id           INT (PRIMARY KEY, AUTO_INCREMENT)
   • nome         VARCHAR(255) - Nome do usuário
   • email        VARCHAR(255) - Email único
   • senha        VARCHAR(255) - Senha criptografada
   • tipo         ENUM('user', 'admin') - Tipo de conta
   • created_at   TIMESTAMP - Data de criação

Execute o arquivo database_setup.sql para criar a tabela automaticamente.

🧪 COMO TESTAR
═══════════════════════════════════════════════════════════════════════════════

PASSO 1 - CRIAR CONTA:
   1. Acesse http://localhost:8080/ControleExterno/
   2. Clique em "Cadastre-se aqui"
   3. Preencha os dados:
      • Nome: Seu Nome Completo
      • Email: seu@email.com
      • Tipo: Usuário (ou Administrador)
      • Senha: Uma senha com mínimo 6 caracteres
   4. Confirme a senha
   5. Clique em "Criar Conta"

PASSO 2 - FAZER LOGIN:
   1. De volta ao login, preencha:
      • Escolha o tipo de login (Usuário ou Admin)
      • Email: o email cadastrado
      • Senha: a senha criada
   2. Clique em "Entrar"

PASSO 3 - EXPLORAR:
   1. Você será redirecionado para o dashboard
   2. Clique em "Usuários" para ver a lista
   3. Explore as funcionalidades
   4. Clique em "Sair" para fazer logout

📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

❌ "Erro no banco: Unknown database 'ControleExterno'"
   → Crie o banco: CREATE DATABASE ControleExterno;

❌ "Erro no banco: Table 'Users' doesn't exist"
   → Execute o arquivo database_setup.sql no seu MySQL

❌ "Email ou senha inválidos"
   → Verifique se digitou corretamente
   → Certifique-se de que cadastrou a conta

❌ "Email já cadastrado"
   → Esse email já existe no sistema
   → Use outro email ou faça login com este

❌ Página em branco
   → Verifique os logs de erro do PHP
   → Verifique a conexão com o banco de dados
