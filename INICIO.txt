🧪 TESTE O SISTEMA - PASSO A PASSO:

   □ Preencha: Nome, Email, Tipo, Senha
   □ Clique em "Criar Conta"

   □ Acesse index.php

   □ Como admin, veja o "Painel Administrativo"

PASSO 4: Fazer logout
   □ Clique em "Sair" no canto superior direito
   □ Você será deslogado automaticamente
   □ Será redirecionado para o login

💡 DICAS IMPORTANTES:
═══════════════════════════════════════════════════════════════════════════════

1. Emails devem ser únicos → Cada email só pode cadastrar uma vez
2. Senhas são criptografadas → Não é possível recuperar senha original
3. Tipo é determinado no banco → Mude via SQL: UPDATE Users SET tipo='admin'
4. Sessão por navegador → Fecha ao fechar o navegador
5. Logout destrói tudo → Nenhum rastro fica na sessão

═══════════════════════════════════════════════════════════════════════════════
🆘 PROBLEMAS COMUNS:
═══════════════════════════════════════════════════════════════════════════════

❓ "Erro no banco de dados"
   → Verifique se MySQL está rodando
   → Verifique se criou o banco ControleExterno
   → Verifique as credenciais no código

❓ "Email ou senha inválidos"
   → Verifique a digitação
   → Certifique-se que tem conta cadastrada
   → Email e senha são case-sensitive

❓ "Email já cadastrado"
   → Use outro email
   → Ou faça login com este email

❓ "Página em branco"
   → Verifique os logs do PHP
   → Abra o console do navegador (F12)
   → Procure por erros JavaScript
